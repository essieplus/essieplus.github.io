个人向-博客写作指南

2022-07-11

给自己写说明书

自己架设博客的工作流是比较复杂的，相信在写作过程中也会不断的迭代升级，为了防止遗忘，建立升级的baseline，还是决定给自己写一篇继续写作的指南。

解决的问题

  • 写作与发布的快速试错与迭代
  • 写作与博客分离
    • Bluehost 中 WordPress 主题不好编辑
    • 使用简单的 Markdown 风格
